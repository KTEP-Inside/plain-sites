Тестирование программного обеспечения

Автоматизированное тестирование


Список задач, решить которые помогает автоматизация:

- выполнение тест-кейсов, непосильных человеку;

- решение рутинных задач;

- ускорение выполнения тестирования;

- высвобождение человеческих ресурсов для интеллектуальной работы;

- увеличение тестового покрытия;

- улучшение кода за счёт увеличения тестового покрытия и применения специальных техник автоматизации.

Категории тестов, которые не подходят для автоматизации:

- тестовые случаи, которые были разработаны и не выполнялись вручную хотя бы один раз;

- тестовые случаи, для которых часто меняются требования;

- тестовые случаи, которые выполняются на разовой основе.
