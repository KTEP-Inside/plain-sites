Создание макета страницы для веб–сайта


23. Далее требуется написать на «кнопке» необходимый текст (метку), который будет подсказывать пользователям, каким образом можно дальше продолжать просматривать через веб–страницу очередные проекты.

24.  Следует обозначить конец содержимого на экране областью нижнего колонтитула. Это можно сделать при помощи новой прямоугольной области, создать которую на новом слое и залить светло-серым цветом.

25. Инструментом Elliptical  Marquee Tool «Овальное выделение» на новом слое нужно изобразить длинный и узкий эллипс и заполнить это выделение радиальной градиентной заливкой цветом от черного к прозрачному. При активном слое с нарисованным эллипсом, нужно нажать комбинацию клавиш , либо: Edit>Transform> Scale (Редактирование – Трансформирование – Масштабирование) для того, чтобы созданный эллипс масштабировать в длинный и тонкий эллипс, похожий на тень.

26. Далее разместить эллипс - тень на экране по центру, затем удалить лишнюю область выше нижнего колонтитула. В результате, должна получиться тонкая тень, которая добавит небольшой объем деталям.

27. Область нижнего колонтитула содержит достаточно свободного места для размещения вторичных элементов страницы. Одним из таких элементов может стать «Область клиентского логина». Для того, чтобы создать данную область, требуется активизировать инструмент Type «Текст», создать два (или несколько) текстовых рамок, внутри которых и разместить текст: логин; пароль. Созданным текстовым рамкам можно придать небольшую мягкую тень, используя  функцию: Layer Style «Стили слоя».

28. Следует использовать центральную область нижнего колонтитула для того, чтобы разместить информацию о компании. На этом этапе создается текст, используя легко читаемый шрифт.

29. Теперь, нужно добавить контактную информацию крупным читаемым шрифтом.

30. В итоге, после всей проделанной работы, получилась интересная веб – страничка. Все элементы четко расположены на странице, все удобочитаемо и красиво!
