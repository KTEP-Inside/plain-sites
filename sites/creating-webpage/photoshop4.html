13. Переход к боковой панели. Для начала нужно создать и сохраним отдельным файлом градиентную заливку для её заголовков. С инструментами действовать также, пояснения не требуются.

14. Добавить на только что созданный градиент текст-заголовок информационного блока.

15. Теперь нарисовать обрамление области. Для этого достаточно использовать прозрачный прямоугольник с чёрными линиями контура. Нужно выбать инструмент Прямоугольник, задать тип заливки фигуры Нет цвета, щёлкнуть значок Задать тип штриха фигуры и выбрать чёрный цвет, иначе линий не будет. Если контур получился слишком толстым, устанавить ширине линий значение 0,5 пт.

16. Ниже добавить заголовок блока с градиентной заливкой.

17. Создание блока меню левой панели. Добавить оранжевый прямоугольник шириной 100px с жёлтым контуром 0,2 пт.

18. Добавить на него текст.

19. С помощью дублирования слоёв и линеек перейти к созданию ещё пяти пунктов меню левой панели.
